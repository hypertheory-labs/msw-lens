# .msw-lens/

This directory is owned by msw-lens. Files here are generated — do not edit manually.

## Files

### context.md
Always-current snapshot of this project's mock state: all endpoints, all scenarios,
which is currently active. Regenerated every time `npm run lens` is run.

**Drop this into any LLM conversation for instant context about what is mocked here.**

### prompt.md
Generated on demand by `npm run lens:context -- <component-file>`. Contains the full
source of the component and its dependencies, plus a pre-written prompt asking an LLM
to generate manifests and handler stubs for it.

**Paste this into any LLM conversation to get scenario suggestions for a component.**
